# Симулятор ЧМ-2026

Интерактивный симулятор чемпионата мира по футболу 2026 года с правильным регламентом (48 команд, 12 групп, стыковые матчи, правильная сетка плей-офф).

## 🎯 Текущий статус

**Рабочая версия** (`index.html` + `app.js` + `style.css` + `data.js`) - полностью функциональный прототип с режимами "Нормально" и "Сложно".

## 📁 Структура проекта

### Корневые файлы (текущая рабочая версия)
- `index.html` - главная страница приложения
- `app.js` - основная логика (2081 строка, будет рефакторен)
- `style.css` - стили
- `data.js` - данные турнира (группы, команды, стыковые матчи, сетка)

### `src/` - новая модульная структура (в разработке)

#### `src/core/` - ядро приложения
- `state.js` - управление состоянием и localStorage
- `qualifiers.js` - логика выхода из групп и определения 32 участников плей-офф
- `bracket.js` - построение сетки плей-офф
- `teams.js` - справочник команд, флаги, отображение

#### `src/modes/` - режимы сложности
- `normal.js` - режим "Нормально" (drag&drop групп, выбор 8 третьих, клики по плей-офф)
- `hard.js` - режим "Сложно" (полный ввод счётов, таблицы, статистика)

#### `src/targets/` - точки входа для разных платформ
- `dev-app.js` - версия для разработки/песочницы (текущий `app.js`)
- `sportsru-widget.js` - виджет для встраивания в медиа (sports.ru)
- `telegram-miniapp.js` - Telegram Mini App (для "Мяч продакшен")
- `yandex-game.js` - версия для Яндекс.Игры

#### `src/integrations/` - интеграции с внешними API
- `telegram-sdk.js` - обёртка для Telegram WebApp API

## 🚀 Планы развития

1. **Рефакторинг** - перенос логики из `app.js` в модули `src/core/` и `src/modes/`
2. **Telegram версия** - адаптация под Telegram Mini App
3. **Версия для медиа** - упрощённый виджет для встраивания
4. **Яндекс.Игры** - полная версия со всеми режимами

## 📋 Документация

- `PRESENTATION_TIPS.md` - советы по презентации продукта потенциальным покупателям

Просто открой `index.html` в браузере. Всё работает локально, без сервера.
