# Интеграции

Папка для интеграций с внешними платформами и API.

## Файлы

- `telegram-sdk.js` - обёртка для Telegram WebApp API (для версии под "Мяч продакшен")

## Будущие интеграции

- `sportsru-api.js` - интеграция с API sports.ru (если понадобится)
- `yandex-games-sdk.js` - интеграция с Яндекс.Игры SDK (для версии под Яндекс Игры)
